轻量但够用

只围绕真实 userscript 的核心路径：安装、匹配、运行、启停、调试。没有在线市场和云同步，也不把自用工具做成笨重平台。

为 Safari 留后路

基于 Apple 支持的 Safari Web Extension 与 Manifest V3，用 Xcode 打包成 macOS App，让脚本管理器能稳定留在 Safari 里。

看得见的控制

脚本源码、匹配规则、GM 存储和运行错误都能在管理页检查。出问题时，先知道是谁在什么时候失败。

自己打包，自己掌控。

$ git clone https://github.com/QingZzao/userscript-manager-safari-extension.git
$ cd userscript-manager-safari-extension
$ ./scripts/build-and-install.sh
